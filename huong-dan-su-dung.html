1. Tổng quan

App này dùng để hỗ trợ thao tác tự động với app VPBank trên điện thoại, trọng tâm hiện tại là:

  • Tự detect app VPBank trên máy Android.
  • Mở app VPBank bằng nút VPBank với retry nội bộ để tăng độ ổn định.
  • Chạy luồng Auto All với danh sách MKH.
  • Resume danh sách MKH và vị trí đang chạy dở.
  • Hiển thị trạng thái, log sự kiện, và cảnh báo bảo mật.
Hiện tại bản mới đã có thể mở app VPBank end-to-end trên máy test. Phần login hoặc flow nghiệp vụ sâu hơn có thể tiếp tục tối ưu sau.

2. Chuẩn bị trước khi cài

Hạng mục Yêu cầu
Hệ điều hành PC Windows 10/11
JDK JDK 17
Android SDK Có platform-tools và Build Tools phù hợp
Android Studio Khuyên dùng để mở project và build
Điện thoại Android có cài sẵn app VPBank
Nếu chỉ cần cài bản build sẵn cho bạn bè, có thể bỏ qua phần build source và dùng APK đã build sẵn nếu bạn cung cấp riêng.

3. Clone repo

  1. Mở terminal

    Dùng PowerShell, CMD hoặc terminal trong VS Code.

  2. Clone project
    git clone https://github.com/tatonline2025-prog/VPBankAT.git
cd VPBankAT
  3. Mở project

    Mở thư mục project bằng Android Studio hoặc VS Code.

4. Build APK

Cách 1 - Build bằng Android Studio

  1. Mở project trong Android Studio

    Chờ Gradle sync hoàn tất.

  2. Chọn Build APK

    Vào Build > Build Bundle(s) / APK(s) > Build APK(s).

  3. Lấy file APK

    File APK thường nằm tại:

    app/build/outputs/apk/debug/app-debug.apk

Cách 2 - Build bằng command line

gradlew.bat assembleDebug
Nếu lỗi JDK, hãy bảo đảm máy đang dùng JDK 17 thay vì JDK 21 hoặc JDK 25.

5. Cài APK lên điện thoại

Cách 1 - Cài thủ công

  • Chép file app-debug.apk vào điện thoại.
  • Mở file APK và cho phép cài từ nguồn không xác định nếu máy hỏi.

Cách 2 - Cài qua ADB

adb devices
adb install -r app/build/outputs/apk/debug/app-debug.apk
Muốn cài qua ADB thì phải bật USB Debugging tạm thời. Sau khi cài xong và test detect hoặc mở app VPBank xong thì nên tắt lại để app bank hoạt động đúng điều kiện bảo mật.

6. Cấp quyền cần thiết

  1. Bật Trợ năng

    Vào Cài đặt > Trợ năng > VPBank Controller / VPBankAT > Bật.

  2. Không tắt màn hình khi đang auto

    Trong lúc chạy auto, giữ màn hình sáng và không khóa máy.

  3. Cho phép app không bị kill nền nếu cần

    Một số máy Realme, Xiaomi hoặc Oppo có thể tự tắt app nền quá sớm.

7. Cách dùng app

  1. Mở app

    Màn hình chính sẽ có các khu vực: package VPBank, PIN Smart OTP, danh sách MKH, 3 nút điều khiển, Auto All, trạng thái bảo mật và log sự kiện.

  2. Nhập PIN Smart OTP

    Nhập đúng 6 chữ số. PIN chỉ lưu tạm trong RAM.

  3. Nhập hoặc import danh sách MKH

    Có thể gõ trực tiếp hoặc bấm nút TXT/CSV để import.

  4. Bấm nút VPBank

    App sẽ tự scan candidate, detect package đúng, rồi gửi lệnh mở app VPBank.

  5. Bấm Bắt đầu hoặc Auto All

    Nếu các điều kiện đã đủ, app sẽ bắt đầu luồng tự động.

8. Luồng Auto All

Nút Auto All được thiết kế để gom các bước lại thành một flow:

  1. Check Trợ năng.
  2. Check danh sách MKH.
  3. Check PIN Smart OTP.
  4. Check Developer Options và USB Debugging.
  5. Detect package VPBank.
  6. Mở app VPBank.
  7. Gửi lệnh START cho service automation.
Nếu quay lại app từ màn hình Developer Settings, flow có thể tiếp tục lại mà không cần làm lại toàn bộ từ đầu.

9. Resume danh sách MKH

Bản hiện tại có cơ chế resume:

  • Lưu draft danh sách MKH.
  • Lưu current index đang chạy.
  • Nếu app mở lại mà danh sách không đổi, có thể tiếp tục từ vị trí đang dở.
Nếu đổi hẳn danh sách MKH thì app sẽ chạy lại từ đầu.

10. USB Debug và Developer Options

App VPBank có thể không cho mở hoặc không hoạt động bình thường nếu điện thoại đang bật USB Debugging hoặc Developer Options.

Quy trình đúng để test

  1. Bật USB Debugging tạm thời để cài APK hoặc test bằng ADB.
  2. Test detect hoặc mở app VPBank cho chắc chắn.
  3. Tắt USB Debugging.
  4. Nếu vẫn bị chặn, tắt luôn Developer Options.
  5. Quay lại app và bấm Auto All hoặc VPBank.

App hiện đã có badge trạng thái bảo mật để báo trực tiếp:

  • Dev OFF, USB Debug OFF
  • Dev ON hoặc USB Debug ON

Bước 1 – Cài APK lên điện thoại

  1. Bật cài từ nguồn không xác định
    Android 8+: Cài đặt → Ứng dụng → [File manager] → Cho phép từ nguồn này
    Android 7: Cài đặt → Bảo mật → Nguồn không rõ ràng → bật
  2. Chuyển file APK sang điện thoại
    Qua Zalo, email, cáp USB, hoặc Google Drive. File: app-release.apk (~4.3 MB)
  3. Mở file APK → Cài đặt → Mở
    Xác nhận nếu hệ thống hỏi về quyền truy cập.
⚠️ Không cần USB Debug hay chế độ Developer. APK đã được ký bằng release keystore — VPBank NEO sẽ hoạt động bình thường.

Bước 2 – Bật Accessibility Service

ℹ️ Bước này bắt buộc. App dùng Accessibility Service để điều khiển VPBank NEO thay người dùng.
  1. Vào Cài đặt → Trợ năng (hoặc Accessibility)
  2. Tìm "VPBank Controller" trong danh sách dịch vụ
  3. Nhấn vào → bật công tắc → nhấn OK / Cho phép khi có hộp thoại cảnh báo
✅ Nếu chưa bật, app sẽ tự động mở trang Accessibility Settings khi bạn nhấn "Bắt đầu".

Bước 3 – Chuẩn bị file MKH

Định dạng hỗ trợ

Định dạngYêu cầu
Excel XLSX MKH nằm ở cột A, bắt đầu từ dòng 1 (hoặc dòng 2 nếu có tiêu đề)
CSV / TXT MKH ở cột đầu tiên, phân cách bằng dấu , hoặc ;

Prefix MKH hợp lệ

PrefixLoại điệnVí dụ MKH
PB Điện toàn quốc (EVN) PB1234567, PB9876543
PE Điện Hồ Chí Minh (EVN HCM) PE9876543, PE1111222
Khác Không hỗ trợ Tự động bỏ qua

Ví dụ file CSV

PB1234567 PB2345678 PB3456789 PE9876543 PE8765432
ℹ️ Sau khi import, app tự sắp xếp: PB lên đầu, PE theo sau. Dòng tiêu đề (không có prefix PB/PE) sẽ bị bỏ qua tự động.

Bước 4 – Nhập thông tin trong app

  1. Import file MKH
    Nhấn nút "Ấp TXT/CSV" → chọn file từ bộ nhớ máy.
    App hiện toast xác nhận số lượng: "Đã load N MKH". Kiểm tra danh sách MKH hiển thị trong ô text.
  2. Nhập PIN Smart OTP (6 chữ số)
    Đây là mã PIN bạn tự đặt khi kích hoạt Smart OTP trong VPBank NEO — khác với mã OTP ngẫu nhiên.
  3. (Tuỳ chọn) Package VPBank NEO
    Mặc định: com.vnpay.vpbankonline. Không cần thay đổi.

Bước 5 – Chạy tự động

  1. Mở VPBank NEO → vào Thanh toán hóa đơn → Điện
  2. Chuyển lại VPBank Controller
  3. Nhấn "Bắt đầu"

App sẽ tự động thực hiện với mỗi MKH:

  • Chọn "Điện toàn quốc" hoặc "Điện Hồ Chí Minh" theo prefix PB/PE
  • Nhập mã khách hàng vào ô MKH → bấm Tiếp tục
  • Chờ hiện thông tin hóa đơn → bấm Tiếp tục
  • Bỏ popup "Kích hoạt thanh toán tự động" (nếu xuất hiện)
  • Bấm Xác nhận thanh toán
  • Tự động nhập PIN Smart OTP 6 chữ số
  • Bấm "Xác nhận giao dịch"
  • Chờ màn hình "Thành công"
  • Bấm "Giao dịch mới" → chuyển sang MKH tiếp theo
